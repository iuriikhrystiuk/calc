Rules:

* currently supported operators: +, -, *, /, %, //, (, )
* '//' operator converts quotient to integer
* '%' operator calculates remainder of the division
* number decimal symbol is '.'
* you can generate formulas that include number and identifiers
* indentifiers accept other formulas
* indentifiers can be function
* missing indentifiers from all formulas are combined in one list

Supported Functions:

* gt(a,b): returns 'true' if a > b, 'false' otherwise
* gte(a,b): returns 'true' if a >= b, 'false' otherwise
* lt(a,b): returns 'true' if a < b, 'false' otherwise
* lte(a,b): returns 'true' if a <= b, 'false' otherwise
* eq(a,b): returns 'true' if a = b, 'false' otherwise
* if(a,b,c): returns 'b' if 'a' is 'true', 'c' otherwise
* num(a): converts 'a' to a number
